Task Manager Web Application
Project Description

This project is a Task Manager Web Application built using the Django framework. The application allows users to register, authenticate, and manage tasks efficiently through a simple and secure interface.

The system demonstrates modern DevOps practices, including containerization with Docker, version control using GitHub, and automated deployment using a CI/CD pipeline. The application is deployed on a remote server and can be accessed via HTTPS.

The main goal of this project is to showcase the integration of web development and DevOps workflows, including Docker-based deployment and continuous integration.

Features

- User authentication (login and registration)

- Create new tasks

- Update existing tasks

- Delete tasks

- View task list

- Persistent database storage

- Dockerized application

- Automated deployment with GitHub Actions

- Secure production deployment with Nginx
